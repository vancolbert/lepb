# lepb [<img src="https://github.com/vancolbert/trucsle/raw/main/flag-fr.svg" width="32" height="20" align="right"><img src="https://github.com/vancolbert/trucsle/raw/main/lang-fr.svg" align="right">](README.fr.md)
Blender import/export addon for [Landes Eternelles](http://www.landes-eternelles.com/) game assets:
- 3D object meshes (e3d)
- 2D ground decals (2d0)
- Maps (elm)
- Cal3D meshes (cmf)
- Cal3D skeletons (csf)
- Cal3D animations (caf)
- Model definitions (actor_defs xml)

Tested on Blender 5.0 with Python 3.11.13 and numpy 1.26.4. Older versions may require workarounds.
